# Landing page ad variant copy

**Prompt Name:** Landing page ad variant copy  
**Category:** Ad copy  
**Persona/Role:** Conversion copywriter  
**Goal:** Create ad copy variants that align tightly with a landing page offer.  
**Prompt Text:**

```text
Context: We are promoting [OFFER] for [PRODUCT NAME]. The landing page promise is [LANDING PAGE PROMISE]. The audience is [AUDIENCE]. Approved claims are [APPROVED CLAIMS].
Objective: Write ad variants that match the landing page message and reduce post-click mismatch.
Style: Specific, conversion-focused and credible.
Tone: [TONE].
Audience: [AUDIENCE].
Response: Return 6 variants with Primary Text, Headline, Description, CTA, Landing Page Message Match and Risk Note. Do not introduce benefits that are absent from the landing page.
```

**Variables/Placeholders:** `[OFFER]`, `[PRODUCT NAME]`, `[LANDING PAGE PROMISE]`, `[AUDIENCE]`, `[APPROVED CLAIMS]`, `[TONE]`  
**Example Output:**

| Primary Text | Headline | Description | CTA | Landing Page Message Match | Risk Note |
|---|---|---|---|---|---|
| Build a cleaner close-week reporting workflow with a checklist made for SaaS finance teams. | Close-week checklist | Practical steps for recurring revenue reporting. | Download checklist | Matches checklist offer and finance audience | Does not imply software purchase is required |
| Revenue reporting gets easier when owners, account movement and next steps are visible before close. | Improve close readiness | See the workflow checklist. | Get the guide | Mirrors the landing page promise | Avoids unsupported time savings |

**Best Practices/Tips:** Compare each ad to the landing page before launch. Keep the offer, audience and claim language consistent. Use the risk note to catch unsupported claims before media spend starts.
